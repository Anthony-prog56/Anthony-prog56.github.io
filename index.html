🕷️ Arac Attack

Engine: Unity – C++

This project is based on a 2D platformer template where the player controls a soldier fighting spiders to reach the end of the level. The base version allowed movement, shooting in all directions, and enemies that moved between waypoints and caused contact damage. I added a new flying enemy with improved AI using a behavior tree, implemented a grappling hook for the player, added moving platforms, and developed most of the HUD (except the health bar).

🔗 Watch Demo | GitHub Repository

♟️ Your Turn

Engine: Unreal Engine – C++

"Your Turn" is a prototype of a turn-based game where the player controls multiple characters on a grid-based map similar to a chessboard. AI enemies act intelligently based on the situation using a behavior tree. Each character and enemy has their own inventory, weapons, and abilities.

🔗 Watch Demo | GitHub Repository
